# Etik Kurul Yönetim Sistemi

Antalya Belek Üniversitesi için geliştirilen, akademik etik başvuru süreçlerini dijitalleştiren web tabanlı yönetim sistemi.

## Kullanılan Teknolojiler
- **Backend:** PHP
- **Veritabanı:** PostgreSQL (Neon Tech Cloud)
- **Arayüz:** HTML5, CSS3, Bootstrap

## Özellikler
- Rol Bazlı Erişim Kontrolü (RBAC)
- İş Zekası (BI) ile başvuru raporlama
- Veri Bütünlüğü Koruması
